##Сервис автоматического обновления баз 1С. 

Шаги по обновлению вынесены во внешний json-файл. Задания на обновления поступают от github.
Первоначально производится обновление тестовой базы, дымовое тестирование измененных объектов. 
При измененении общих модулей, используя граф связности, определяется список связанных объектов для тестирования.
Изменение прямых объектов, добавляется в список автоматом.
При положительном результате тестирование, производится обновлениме прод-базы.
Все шаги детельно логируются. 
Простой web-доступ, для управления сервиса, мониторига задач.
