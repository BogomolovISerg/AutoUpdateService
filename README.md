# Сервис автоматического обновления баз 1С. 

Сервис предназначен для автоматизации обновления баз 1С.
Сценарии обновления вынесены во внешний JSON-файл, что упрощает настройку и сопровождение.
Задания на обновление поступают из GitHub.

Обновление начинается с тестовой базы, после чего выполняется дымовое тестирование изменённых объектов.
Если изменяются общие модули, с помощью графа связности формируется список связанных объектов, подлежащих тестированию.
Изменённые прямые объекты включаются в этот список автоматически.

При успешном завершении тестирования выполняется обновление рабочей базы.
Все этапы работы сервиса подробно логируются.
Для управления сервисом и мониторинга задач предусмотрен простой web-интерфейс
